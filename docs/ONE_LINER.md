# One-liner

OfficeWiki는 조직의 Raw 자료(웹/이미지/문서/폴더)를 workspace에 넣기만 하면 Parsed/Indexed Markdown으로 정리되고, 그 위에서 Wiki와 Ontology(거버넌스)를 만들며, 결과적으로 agent workflow를 운영할 수 있게 하는 **LLM‑Wiki Enterprise** 레이어다.

## NOT
- 단순 위키(Confluence/Notion) 대체가 목적이 아니다.
- Ontology를 ‘처음부터 완성된 모델로 설계’하는 탑다운 프로젝트가 아니다.

## First user scenarios
1) 공유 폴더에 자료를 던져 넣으면, 자동으로 Markdown wiki가 생성되고 검색/드릴다운이 된다.
2) Wiki를 기반으로 공통 용어/규칙/워크플로우 후보가 추출되고, 사람이 승인하여 Ontology로 승격한다.
