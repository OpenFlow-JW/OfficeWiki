# OfficeWiki

LLM‑Wiki Enterprise: 조직의 Raw 자료를 ingest해 Parsed/Indexed Markdown → Wiki → Ontology(거버넌스)로 승격시키고, 그 위에서 agent workflow를 생성/운영하는 **Wiki-based IDE**.

## Core pipeline (v0)
1) Raw (`workspace/raw`)
2) Parsed/Indexed Markdown (`workspace/wiki`)
3) Agent Workflows (`workspace/workflows`)
4) Governance Ontology (`workspace/ontology`)
